# Discord Automation Bot

A simple Discord raider tool.

## Setup

1. Create a file named `tokens.txt` and paste one token per line.

```
token1
token2
token3
```

2. Copy the code into a file named `main.py`.

3. Run:

```bash
python main.py
```
